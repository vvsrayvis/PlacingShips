Simple SPA with drop area where to ships can be dragged. 
Area of dock and ships received from API. 
Used language's HTML, CSS, C# .Net 9 Framework, JS
To run test use command: 
cd path/to/solution-or-test-project
dotnet test.
Or just runTest from Visual studio.
It is possible to rotate ships 90 degrees to utilize all space in the anchorage. Step over the ship image and press "space" button
Overlap between vessels is not possible
Ship can't overlap with area borders.
For use the same set of ships use button "Try again" on a screen
